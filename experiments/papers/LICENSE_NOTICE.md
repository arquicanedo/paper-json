# License Notice — Experiments Corpus

The five subdirectories under `experiments/papers/` contain materials derived
from third-party academic papers hosted on arXiv. They are used here as a test
corpus for the PROSE-vs-JSON pilot described in the genesis paper (`../../paper.typ`,
claims C1 and C5).

## What is here

For each arXiv paper:

- `paper.json` — a structured re-encoding of the paper's claims and definitions
  produced for benchmarking purposes. Authored by this project; derivative of
  the source paper.
- `ground_truth.json` — scoring key for the pilot, hand-curated by this project.
- `paper.pdf` — local copy of the published arXiv PDF. **Not redistributed via
  this repository** (gitignored). Regenerate locally:
  `uv run experiments/regenerate_papers.py`
- `paper.txt` — plain-text extraction of the PDF, used by the PROSE-condition
  runner. **Not redistributed** (gitignored). Regenerated by the same script
  via `pdftotext` (poppler-utils).
